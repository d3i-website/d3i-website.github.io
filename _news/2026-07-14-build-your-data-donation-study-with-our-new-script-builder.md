Researchers using Port for data donation no longer need to write their own Python script for local data processing. Our new Script Builder makes setting up a study much easier.

The Script Builder lets researchers create a Python script tailored to their study through a simple, user-friendly interface. Just select the platform you're interested in and the data you want to collect, and the tool will generate the script for you.

At the moment, the Script Builder supports nine platforms in both Dutch and English. We're also working on several new features, including visualizations and an automatic codebook generator, to make preparing data donation studies even more straightforward.

Give the Script Builder a try, and let us know what you think. If you have feedback or ideas for new features, we'd be happy to hear them.
