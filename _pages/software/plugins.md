## Make Your Study More Interactive

Port includes optional features (plugins) that let you create richer, more dynamic experiences for participants. These features go beyond basic data extraction to create truly interactive studies.

## Available Features

### Interactive Questionnaires

Ask participants questions about their donated data in real-time. This feature allows you to use the actual donated data as input to your questions, creating a personalized experience.

**Example use cases:**

- Show participants specific Facebook groups they're members of and ask why they joined
- Display their most-watched Netflix shows and ask about viewing preferences
- Present their Instagram followers and ask about relationship types
- Show WhatsApp message patterns and ask about communication habits

The questionnaire appears during the donation process, after data extraction but before final consent. This means participants can reflect on their specific data while answering.

<img src="/assets/images/questionnaire.png" alt="Interactive questionnaire showing personalized questions based on donated data" style="width: 600px; max-width: 100%;">

**Benefits:**

- Richer contextual data linked directly to behavior
- More engaging participant experience
- Reduced need for separate survey tools
- Real-time validation and follow-up questions

### Multiple File Upload

Some platforms or studies require participants to upload more than one file. The multiple file feature makes this seamless.

**Example use cases:**

- Multiple WhatsApp group chat exports
- Data from different time periods
- Separate files from related platforms
- Different data types from the same platform

Participants see a clear interface showing which files are needed and which have been uploaded successfully.

<img src="/assets/images/multiple_file_screen.png" alt="Multiple file upload interface showing file selection and status" style="width: 600px; max-width: 100%;">

**Benefits:**

- Support for complex, multi-file data donations
- Clear progress indication for participants
- Individual validation of each file
- Flexible data collection scenarios

## Using Features in Your Study

These features are built into Port and can be activated when you configure your data donation task. The technical setup is documented in our [developer documentation](https://d3i-infra.github.io/data-donation-task/).

If you're not comfortable with the technical implementation, our team can help set up these features for your specific study. [Get in touch](/software/help/) to discuss your needs.

## Combining Features

You can use multiple features together. For example:

- **Multiple files + questionnaires** - Collect several WhatsApp chats and ask about each one
- **Visualizations + questionnaires** - Show participants graphs of their data, then ask interpretive questions
- **All features** - Create comprehensive, highly interactive donation experiences

The flexibility of these features allows you to design studies that precisely match your research questions.
