Pick the data you need from a platform (for example, Instagram likes and followers), and the builder packages a tailored Python script for your study.

{% include script-builder-embed.html %}

## About the script you get

The builder composes scripts from the same set of [ready-made platform scripts](/software/ready-made-scripts/), packaged into a single Python wheel for what you picked. Drop the wheel into your data donation study like any other script. Extracted data stays on the participant's device until they consent to donate it.
